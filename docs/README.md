# Documentation

This folder contains project documentation for both users and developers.

## Documents

- `docs/user-guide.md` — day-to-day usage for staff roles.
- `docs/developer-guide.md` — local setup, architecture, and operations.
- `docs/user-developer-guide.md` — combined guide for quick onboarding.

## Quick Links

- Backend API docs (when running locally):
  - Swagger UI: http://localhost:8000/docs
  - ReDoc: http://localhost:8000/redoc

## Conventions

- Paths are relative to the repository root.
- Environment variables are configured in `.env` (see `.env.example`).
- Backend runs on `http://localhost:8000`, frontend on `http://localhost:5173` by default.
